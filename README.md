# 🛡️ Omniscope
**Interactive Automated Reconnaissance Framework**

Omniscope is a powerful Bash-based tool designed for Bug Bounty hunters and Security Analysts to automate the initial phases of reconnaissance.

## 🚀 Key Features
- **Subdomain Enumeration** (Subfinder)
- **Live Host Probing** (Httpx)
- **Vulnerability Scanning** (Nuclei)
- **Port Auditing** (Nmap)
- **Tech Stack Identification** (WhatWeb)

## 🛠️ How to Run
1. `git clone https://github.com/moez-io/omniscope.git`
2. `cd omniscope`
3. `chmod +x omniscope.sh`
4. `sudo ./omniscope.sh`
